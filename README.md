# IPL Crushes — Wooble Crunch-26 Hackathon Submission

An interactive web application built for the **[Wooble Crunch-26 Hackathon](https://wooble.org/hackathons)**. This project blends the massive craze of the Indian Premier League (IPL) with a fun, gamified social concept, allowing users to discover, vote on, and rank their favorite IPL personalities, anchors, or trending moments.

The entire application logic is developed directly in a Google Colab environment and engineered to deploy seamlessly on **Streamlit Community Cloud**.

---

## 🚀 About the Project

This repository contains the official submission for the Wooble Crunch-26 Hackathon. Built using Python and Streamlit, the app focuses on rapid data handling, clean UI presentation, and an engaging user experience without requiring a complex multi-file backend.

### Core Features

* **Interactive Voting Layout:** A sleek, user-friendly card interface to vote on your favorite IPL personalities.
* **Live Leaderboard:** Real-time data processing to showcase top trending IPL crushes based on total votes.
* **Filter by Categories:** Easily sort through players, commentators, anchors, and viral stadium content creators.
* **Data-Driven Insights:** Simple analytics and visualizations embedded directly within the Streamlit dashboard.

## ⚙️ How to Run & Deploy

### Option 1: View / Run in Google Colab

If you want to experiment with the code or run it locally in an interactive environment:

1. Open the `.ipynb` or `.py` file included in this repository.
2. Run the cells to install necessary packages (`pip install streamlit`).
3. Use a tunneling tool like LocalTunnel or Ngrok within Colab to view the live Streamlit server layout.

### Option 2: Deploy directly to Streamlit Cloud

To launch this project into production:

1. Fork or clone this repository to your GitHub account.
2. Go to **[Streamlit Community Cloud](https://share.streamlit.io/)** and log in with your GitHub account.
3. Click **"New app"**, select this repository, choose the main file, and click **"Deploy!"**
